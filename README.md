Queries chatgpt through voice commands

## About
Queries the chatgpt turbo api with the string stated after "ask chatgpt" and responds with chatgpt's response

## Examples
* "Ask chat g p t {query}"
* "Ask chatgpy {query}"
* "Ask chatgpt {query}"
* "Ask chat gee pee tee {query}"
